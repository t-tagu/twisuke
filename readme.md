# ツイ助

ツイッターの相互フォロワー間のスケジュール調整を効率的におこなうサービスです。
イベント作成からDMでのスケジュール調整URLの共有までをまとめて完了することができます。

# 使用フレームワークや言語、ツールなど

Vue.js
Laravel 5.8
SASS
javascript
PHP
HTML5
CSS3
WEBPACK
Gulp
           
# 機能的特徴

* SPAで作成
* Vuexによるログイン状態管理
* ツイッターアカウントログイン
* TwitterAPIによるDM送信機能
